2. **ownerOf**: Confirms that the owner of the token ID is returned correctly and is not the zero address.

3. **approve**: Verifies that the approved address for the token ID is updated correctly, or remains unchanged if the operation fails.

4. **getApproved**: Ensures that the approved address for a given token ID is returned correctly.

5. **setApprovalForAll**: Checks that the approval status for the operator is updated correctly or remains unchanged if the operation fails.

6. **isApprovedForAll**: Confirms that the approval status for the operator is returned correctly.

7. **transferFrom**: Validates that the ownership of the token is transferred correctly, and updates the owned tokens count for both the sender and the recipient.

8. **safeTransferFrom (overloaded)**: Similar to `transferFrom`, but additionally ensures that the token transfer adheres to the ERC-721 rules regarding contract recipients.

9. **safeTransferFrom (with data)**: Same as the previous `safeTransferFrom` but allows for additional data to be sent, verifying that the token ownership is transferred correctly.
